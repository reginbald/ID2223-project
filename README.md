# ID2223-project

Create container:
`docker run -p 8888:8888 --name tensorflow-id2223 -v /path/to/ID2223-project:/notebooks -it gcr.io/tensorflow/tensorflow`

Run container:
`docker start --interactive tensorflow-id2223`
